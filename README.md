
## ant-colony-optimization

Draw arbitrary points on the board and let the üêú "ant colony optimization" algorithm find and display the shortest path crossing all of your points.

This is known as the [Travelling salesman problem](https://en.wikipedia.org/wiki/Travelling_salesman_problem).
